Game Name: Lineage
Author Name: Quang Vo
"A simple almost roguelike RPG game with randomly generated levels, tile and sprite-based graphics, and character perma-death."
Controls:
Up, Down, Left, Right: Move Character
Select: Restart Game
Start: Start Game

Objective: Collect the key, dodge the monsters, and get to the exit!
The game will progressively get harder, see how far you can make it.
On death, you have to option to accept your death and have a true game over, or restart the level.
